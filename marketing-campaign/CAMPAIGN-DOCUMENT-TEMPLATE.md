## Document Format Guidelines

### For Best Auto-Extraction Results:

1. **Use Clear Labels**: Use keywords like "Campaign Name:", "Product:", "Features:", etc.
2. **Separate Sections**: Use blank lines or headers to separate different sections
3. **List Format**: Use bullet points (-) or asterisks (*) for features and goals
4. **Date Format**: Use YYYY-MM-DD format (e.g., 2026-03-01)
5. **Channels**: Mention "email", "sms", or "instagram" explicitly
6. **Budget**: Include $ symbol or write "Budget: 5000"
7. **Contact Info**: Include email addresses, phone numbers, and website URLs

### Supported File Formats:
- PDF (.pdf)
- Microsoft Word (.docx)
- Plain Text (.txt)

## Tips for Auto-Execution

1. **Complete Information**: Include all required fields for best results
2. **Review After Import**: Always review auto-populated data before executing
3. **Add Contact Lists**: Upload contacts separately or select existing list
4. **Channel Selection**: Explicitly mention which channels to use
5. **Personalization**: The system will automatically add personalization to emails

---

## How to Use

1. **Create Your Document**: Use this template or create your own with the required fields
2. **Save as PDF/DOCX/TXT**: Choose your preferred format
3. **Upload in Campaign Creator**: Click "Choose File" on the campaign creation page
4. **Review & Edit**: System auto-fills the form - review and adjust as needed
5. **Select Recipients**: Choose contact list or enter single contact
6. **Execute**: Submit to launch your campaign!
